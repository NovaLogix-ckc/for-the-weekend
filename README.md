# Workplace Survival Kit

A collection of 10 completely essential workplace tools that no office should be without. Built with React, Vite, and questionable life choices.

**Live site:** [novalogix-ckc.github.io/for-the-weekend](https://novalogix-ckc.github.io/for-the-weekend/)

## The Tools

| Tool | What it does |
|------|-------------|
| **Is It Friday Yet?** | Answers the only question that matters. Monday gets the response it deserves. |
| **Excuse Generator** | Produces corporate-grade alibis for tardiness. Assembled from real buzzwords and fake sincerity. |
| **Useless Dark Mode Toggle** | Switches between two shades of off-white. Both are "dark mode." |
| **Apology Email Escalator** | Type a minor mistake. Click escalate 5 times. End up in a cabin with squirrels. |
| **Loading... Forever** | A beautiful progress bar for absolutely nothing. Resets at 92%. |
| **Is My Code Good?** | One button. No input needed. The verdict is final and legally binding. |
| **Meeting vs Email Tribunal** | Describe your meeting. Receive a verdict, a percentage score, and emotional damages. |
| **Passive Aggressive Compliment Machine** | Workplace-safe compliments with plausible deniability. |
| **The Nod Button** | Says "Yep." in a different font every time. Deeply satisfying. |
| **Buzzword Bingo** | Auto-generated bingo card for your next all-hands. Winner leaves early (honour system). |

## Theme

Pip-Boy inspired dark terminal aesthetic with CRT scanline overlay, green-on-black color scheme, and `Share Tech Mono` font. Each page has unique CSS animations. No gradients. No AI slop.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Deployed automatically to GitHub Pages on push to `main`.

## Tech

- React 18
- Vite 6
- react-router-dom 6 (HashRouter for GitHub Pages)
- Pure CSS animations (no libraries)
- Zero API calls - all content generated locally
